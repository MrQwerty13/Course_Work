# Система управления задачами командой (СУЗК)

Это приложение клиент-сервер, разработанное на платформе .NET, предназначенное для улучшения взаимодействия 
и сотрудничества в команде. Система управления задачами команды (СУЗК) позволяет создавать проекты, распределять 
задачи между участниками, отслеживать их выполнение и контролировать сроки на уровне команды.

## Обзор

Система управления задачами команды (СУЗК) предоставляет интуитивно понятный пользовательский интерфейс для работы 
как с клиентской, так и с серверной частью системы. Приложение поддерживает аутентификацию пользователей, 
разграничение доступа и централизованное управление базой данных для обеспечения безопасности и надежности работы 
с данными.
